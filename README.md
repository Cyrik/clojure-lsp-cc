# clojure-lsp-cc

Clojure LSP integration for Claude Code - provides code intelligence for Clojure files.

## Supported Extensions

`.clj`, `.cljs`, `.cljc`, `.edn`

## Prerequisites

Install clojure-lsp: https://clojure-lsp.io/installation/

## Installation

```bash
claude plugin marketplace add github:lukas-domagala/clojure-lsp-cc
claude plugin install clojure-lsp-cc@clojure-lsp-cc
```
